# CONFIG TEMPLATES README

Estos archivos son **PLANTILLAS**. 

## Instrucciones:
1. Copiar el archivo `.example.json` deseado a la carpeta raíz o a la carpeta de ejecución correspondiente.
2. Renombrar eliminando el `.example`.
3. Rellenar con los datos reales (logins, passwords, rutas de terminal).

## Advertencia de Seguridad:
- **NUNCA** hagas commit de los archivos que contengan contraseñas o datos reales.
- El archivo `.gitignore` ya está configurado para ignorar los archivos `.json` que no sean ejemplos.
- Verifica siempre con `git status` antes de un push.
