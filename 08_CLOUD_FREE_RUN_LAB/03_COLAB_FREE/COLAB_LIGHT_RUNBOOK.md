# COLAB_LIGHT_RUNBOOK

## Uso
- Abrir Google Colab.
- Conectar a Drive (opcional, con precaución) o subir archivos directamente a la sesión.
- Instalar dependencias necesarias.
- Ejecutar celdas.

## Recomendación
- Usar solo para validaciones de 10-30 minutos.
- No dejar corriendo toda la noche (se desconecta).
- Ideal para visualizar datos de un output CSV pesado.
