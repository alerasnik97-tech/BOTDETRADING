# BROKER_PROHIBITION_POLICY

- **Cloud free lab jamás conecta broker**: No se permite el uso de librerías de conexión a brokers (MetaTrader API, OANDA API) en instancias gratuitas de nube.
- **Jamás envía órdenes**: El código ejecutado en la nube debe ser 100% offline (backtesting puro).
- **Jamás opera demo/real/fondeo**: Queda prohibido cualquier intento de trading activo en estos entornos.
- **Research Offline Solo**: El uso de la nube se limita exclusivamente a investigación cuantitativa basada en archivos históricos.
