# DATA_PRIVACY_POLICY

- **Mínimo Necesario**: Solo subir la fracción de datos estrictamente necesaria para validar la estrategia.
- **Sin Identificación**: No subir datasets que contengan información personal, nombres de cuentas o históricos de trading real.
- **Datasets Privados**: En plataformas como Kaggle, asegurar que la visibilidad sea siempre "Private".
- **Borrado**: Los datos en la nube deben ser borrados sistemáticamente tras finalizar las tareas.
