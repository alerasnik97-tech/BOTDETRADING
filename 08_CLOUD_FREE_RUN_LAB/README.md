# 08_CLOUD_FREE_RUN_LAB - Preparación para Corridas Gratuitas en Nube

Este laboratorio está dedicado a la preparación, documentación y estrategia de ejecución de corridas de backtesting y research en entornos de nube gratuitos (Oracle Cloud, Kaggle, Colab, GitHub Actions).

## Objetivo
Permitir ejecuciones nocturnas y micro-probes sin costo de VPS, manteniendo la integridad del proyecto y la seguridad de los datos.

## Estructura
- `00_CLOUD_STRATEGY`: Estrategia general y comparativa.
- `01_ORACLE_ALWAYS_FREE`: Guías para Oracle Cloud.
- `02_KAGGLE_NOTEBOOKS`: Guías para Kaggle.
- `03_COLAB_FREE`: Guías para Google Colab.
- `04_GITHUB_ACTIONS_CI`: Automatización liviana.
- `05_CLOUD_PACKAGE_STAGING_NO_ZIP`: Preparación de paquetes (sin comprimir).
- `06_CHECKPOINT_AND_RESUME`: Lógica de recuperación y parada.
- `07_SECURITY_AND_SECRETS_POLICY`: Seguridad y secretos.
- `08_COST_CONTROL`: Control de costos (0.00 USD).
- `09_RUN_TEMPLATES`: Plantillas de ejecución.
- `10_CLOUD_OUTPUT_INBOX`: Bandeja de entrada de resultados.
