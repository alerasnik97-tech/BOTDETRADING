# KAGGLE_OUTPUT_HANDOFF_POLICY

- Los resultados descargados de Kaggle deben ser colocados en `08_CLOUD_FREE_RUN_LAB\10_CLOUD_OUTPUT_INBOX\`.
- Cada descarga debe incluir el `KAGGLE_SMOKE_MANIFEST.json`.
- No se deben subir los ZIPs descargados directamente al repositorio de Git; deben ser descomprimidos y auditados.
- Los logs del notebook de Kaggle deben guardarse como evidencia si ocurre algún fallo.
