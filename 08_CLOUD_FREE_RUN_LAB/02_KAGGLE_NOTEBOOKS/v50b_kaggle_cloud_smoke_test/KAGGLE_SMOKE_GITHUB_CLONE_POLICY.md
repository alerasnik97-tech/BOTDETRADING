# KAGGLE_SMOKE_GITHUB_CLONE_POLICY

- **Fuente de Verdad**: El repositorio en GitHub (`alerasnik97-tech/bottrading`) es la única fuente de verdad permitida.
- **Branch Operativo**: Para esta prueba se utiliza exclusivamente `clean-sync-branch`.
- **Integridad**: Tras el clonado, se debe verificar el último commit mediante `git log` para asegurar que coincide con el estado local esperado.
- **Higiene**: No realizar cambios locales en el repositorio clonado en Kaggle. Cualquier modificación necesaria debe hacerse localmente, subirse a GitHub y re-clonarse en la nube.
