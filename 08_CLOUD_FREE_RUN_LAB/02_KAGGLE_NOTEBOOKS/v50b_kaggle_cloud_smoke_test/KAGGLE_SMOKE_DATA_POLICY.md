# KAGGLE_SMOKE_DATA_POLICY

- **Sin Datos Raw**: Esta prueba NO incluye la subida de archivos `.parquet` o bases de datos históricas pesadas.
- **Sin TEST**: Queda terminantemente prohibido el uso de datos del período de TEST (2025/2026).
- **Finalidad**: La prueba es estrictamente estructural y de entorno (imports y rutas), no de performance o búsqueda de edge.
- **Validación de Datos**: No se valida la calidad de los datos en esta fase.
- **Dataset Privado (Futuro)**: En fases posteriores, se podrá autorizar el uso de un dataset privado en Kaggle con una muestra reducida (p.ej. 1 mes de TRAIN como 2022-05), sujeto a aprobación explícita.
