# KAGGLE_SMOKE_EXPECTED_OUTPUTS

Los archivos esperados tras la ejecución exitosa del Smoke Test en Kaggle son:

1. **KAGGLE_SMOKE_ENV_REPORT.md**: Informe del entorno Python y sistema.
2. **KAGGLE_SMOKE_IMPORT_TEST.txt**: Log de resultados de importación de módulos.
3. **KAGGLE_SMOKE_GIT_REPORT.txt**: Log del estado del repositorio clonado.
4. **KAGGLE_SMOKE_SECURITY_SCAN.txt**: Resultados del escaneo de seguridad preventivo.
5. **KAGGLE_SMOKE_ARTIFACT_PROOF.csv**: El artefacto principal que consolida el éxito de la prueba.

Estos archivos deben descargarse manualmente desde `/kaggle/working/kaggle_smoke_outputs/` y entregarse a Antigravity para su auditoría.
