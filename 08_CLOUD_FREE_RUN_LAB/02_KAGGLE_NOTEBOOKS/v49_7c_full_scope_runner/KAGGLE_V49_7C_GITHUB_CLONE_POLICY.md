# KAGGLE_V49_7C_GITHUB_CLONE_POLICY

- **GitHub es la Fuente de Verdad**: Kaggle debe clonar siempre la última versión del código desde el repositorio oficial.
- **Repo**: `alerasnik97-tech/bottrading`
- **Branch**: `clean-sync-branch`
- **Mecanismo**: Uso de `GH_TOKEN` mediante Kaggle Secrets.
- **Sin ZIP**: No se debe depender de archivos ZIP para la transferencia de código.
- **Netrc**: El archivo `~/.netrc` debe ser borrado tras el clonado exitoso para mayor seguridad.
- **Sincronización**: Los outputs deben ser subidos de vuelta al branch mediante un commit o descargados manualmente para su incorporación controlada.
