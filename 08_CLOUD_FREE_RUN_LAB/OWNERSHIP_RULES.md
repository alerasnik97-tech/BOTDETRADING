# OWNERSHIP_RULES - 08_CLOUD_FREE_RUN_LAB

- Esta carpeta es para preparación de corridas gratuitas en nube.
- No contiene datos crudos.
- No contiene parquet.
- No contiene credenciales.
- No contiene ZIPs de entrega.
- No reemplaza al laboratorio local.
- La fuente de verdad sigue siendo el repositorio local.
- Cualquier corrida cloud debe ser reproducible localmente.
- Ningún output cloud se acepta sin manifest, hash, config y verificación independiente.
