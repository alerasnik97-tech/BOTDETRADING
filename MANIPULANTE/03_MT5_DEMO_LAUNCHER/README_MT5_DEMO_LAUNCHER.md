# MANIPULANTE MT5 DEMO LAUNCHER

Este es un lanzador seguro para abrir el entorno de MANIPULANTE.

## Advertencia Crítica
**DEMO / PAPER ONLY. NO REAL TRADING.**
- MANIPULANTE no permite actualmente operar en dinero real (Real Money Bloqueado).
- No permite el trading algorítmico (No Auto Orders).
- GLOBAL HARD CLOSE FRIDAY 16:55 NY es obligatorio.

## Uso
Puede configurar la ruta a su terminal MT5 Demo en `mt5_path_config.json`. Al ejecutar el bat o ps1, se le mostrarán las advertencias de seguridad y se le abrirán automáticamente los checklists y el archivo de log para que inicie su sesión de manera ordenada.

**Asegúrese de nunca proporcionar las credenciales de una cuenta real o fondeada aquí.**
