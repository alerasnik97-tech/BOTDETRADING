# ESTRATEGIAS - ARCHIVO Y LABORATORIO

Esta carpeta contiene TODAS las estrategias y experimentos que **NO SON LA AUTORIDAD**.

Si usted está buscando qué estrategia operar, diríjase a la carpeta **`MANIPULANTE`** en la raíz del proyecto.

## Contenido de esta Carpeta
- **01_BASELINES**: Estrategias base que sirvieron como pivotes de validación (Phase18, Phase24).
- **02_CANDIDATOS_SHADOW**: Estrategias potentes (como TP1.4 BE0.5) que están en observación paralela, pero que no tienen la máxima autoridad de riesgo.
- **03_RECHAZADAS**: Variantes que fallaron las validaciones de stress (ej. No-BE, WR forzado).
- **04_ARCHIVADAS**: Versiones antiguas que fueron retiradas del entorno productivo (ej. Phase19 por el lookahead bug).
- **05_EXPERIMENTOS**: Pruebas activas o finalizadas sobre la frecuencia o alteración de mecánicas (Phase28, Phase29).
- **06_REPORTES_HISTORICOS**: Copias y referencias a los reportes de las fases iniciales.
- **07_NO_USAR_EN_REAL**: Modelos matemáticos puros que no sobreviven en MT5 o cuentas fondeadas.

**PRECAUCIÓN**: No use NINGUNA de estas estrategias en cuentas fondeadas reales.
